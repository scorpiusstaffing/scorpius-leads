How to install in Gmail

  1. Click ONCE inside the signature you want above (anywhere inside the white box).
  2. Press Cmd + A to select the signature (only the copy-zone is selectable), then Cmd + C.
  3. Open Gmail → ⚙️ → See all settings → General tab → scroll to Signature.
  4. Click + Create new (name it "Scorpius Leads"), or click into an existing one.
  5. Paste with Cmd + V. Logo, fonts, colors and links all carry over.
  6. Under Signature defaults, set "For new emails use" and "On reply/forward use" to this signature for jonathandeklerk@scorpiusleads.com.
  7. Scroll to the bottom and click Save changes.
